一、

```sql
SELECT
	*
FROM
	cn_evaluation_question_option
WHERE
	position(concat(id) IN '1,2,3,4') != 0;
```

二、

```sql
SELECT
	*
FROM
	cn_evaluation_question_option
WHERE
	find_in_set(id, '1,2,3');
```
